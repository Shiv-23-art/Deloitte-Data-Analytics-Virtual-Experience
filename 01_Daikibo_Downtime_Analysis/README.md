# Deloitte Forage – Manufacturing Downtime Analysis

## Overview
Completed as part of the Deloitte Data Analytics Virtual Experience Program (Forage). The objective was to analyze manufacturing telemetry data and identify downtime patterns across factories and device types.

## Tasks Performed
- Created calculated field "Unhealthy"
- Built Downtime per Factory visualization
- Built Downtime per Device Type visualization
- Combined views into an interactive dashboard
- Enabled cross-filtering functionality

## Key Insights
- Certain factories contribute disproportionately to downtime.
- Specific device types show higher failure frequency.
- Interactive filtering allows operational performance comparison across facilities.

## Tools Used
- Tableau
- JSON telemetry dataset

## Deliverable
- Dashboard screenshot demonstrating analysis results
