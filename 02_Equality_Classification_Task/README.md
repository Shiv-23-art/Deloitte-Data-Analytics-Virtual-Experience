# Deloitte Forage – Equality Classification Task

## Overview
This task was completed as part of the Deloitte Data Analytics Virtual Experience Program (Forage). The objective was to classify employee compensation equality scores into predefined categories.

## Dataset Fields
- Factory
- Job Role
- Equality Score (-100 to +100)

## Task Objective
Create a new column called "Equality Class" to categorize scores as:

- Fair (±10)
- Unfair (< -10 OR > 10)
- Highly Discriminative (< -20 OR > 20)

## Approach
- Implemented conditional logic in Excel
- Applied business rules to classify equality scores
- Validated edge-case values

## Tools Used
- Microsoft Excel

## Deliverable
- Edited Excel file with Equality Class column
